### Tips for Recording
*   **Zoom Level:** Ensure the browser window zoom is set appropriately (usually 100% or 110%) so the text in the PDF reader and the tables are readable.
*   **Cursor Tracking:** Move your mouse smoothly. Hover briefly over elements (like the tabs or the highlight markers) before clicking on them so the viewer's eyes can follow.
*   **Snipping Tool Hint:** You can configure the Snipping tool on macOS (using `Cmd + Shift + 5`) to record a selected portion of the screen (just the browser viewport) rather than the whole desktop, keeping the video clean of desktop icons and status bar menus.
